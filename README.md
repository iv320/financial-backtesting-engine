# Financial Backtesting Engine

This project implements a Python-based backtesting system for evaluating trading strategies on historical stock market data.

## Features
- Fetches and cleans financial market data
- Implements rule-based trading strategies
- Backtests strategies on historical data
- Calculates key performance metrics:
  - Profit & Loss (P&L)
  - Maximum Drawdown
  - Sharpe Ratio

## Tech Stack
- Python
- Pandas
- NumPy
- yfinance
- Matplotlib

## Use Case
Designed for quantitative analysis and financial research to support data-driven trading decisions.
